# 一、 前言

使用本产品如果出现无法修正的错误，如果数据已备份或者数据不重要，可使用如下步骤恢复，可能使用到如下工具：

1、 Navicat

2、 DataGrip

3、 PhpMyadmin

在以上工具中任选一项即可，并且准备好产品附带的dump.sql文件。
